  <h1>bazero</h1>
</div>

Zero Config for Babel.

```sh
$ npm i -g bazero
$ bazero
```

bazero uses Babel7.
Input directory is `src`.
Output directory is `dist`.
Default presets are `@babel/preset-env`, `@babel/preset-react` and `@babel/preset-flow`.
